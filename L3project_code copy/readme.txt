This is my L3 project code folder.“L3project copy” contains all of data sets used in the project, the code of handling data preparation and the results of training algorithms“L3project_interface copy” is the project folder for building a website illustrating the accident prevention system. It is developed based on Django. To run the server, type “python manage.py runserver”. It also includes the api keys used for calling api. Tomtom api has daily limited access for free users.
